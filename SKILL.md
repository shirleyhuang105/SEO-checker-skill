# SEO Checker Skill

对网站进行全面的 SEO 健康度检测，生成包含评分、排名表现和改进建议的报告。

## 工作流程

### 1. 获取并解析页面

使用 `web_fetch` 获取 URL 的完整 HTML 内容，保存到临时文件：

```bash
# 保存 HTML 内容到文件
echo "$HTML_CONTENT" > /tmp/page.html
```

### 2. 技术分析

运行检测脚本分析页面技术指标：

```bash
python3 /mnt/skills/user/seo-checker/scripts/seo_analyzer.py "$URL" /tmp/page.html
```

脚本输出 JSON，包含：
- 网站区域识别（`site_region`: china/international）
- 五大维度评分和详细数据
- 问题列表和改进建议（含 `affects_display` 字段）

### 3. 搜索引擎收录检测

根据网站区域使用对应搜索引擎：

**中国站** (`site_region: china`):
- 百度收录：`site:域名 site:www.域名` （百度语法）
- Bing 收录：`site:域名`

**国际站** (`site_region: international`):
- Google 收录：`site:域名`

搜索并记录索引数量估算。

### 4. 关键词排名分析

提取页面的目标关键词（从 title、h1、meta description），搜索这些关键词：

```
"关键词1"
"关键词2" + 域名
```

分析该网站在搜索结果中的排名位置。如果前3页未出现，说明排名较低。

### 5. 外链与权威性

搜索网站的外链情况：

```
"域名" -site:域名 (backlinks OR 外链 OR 链接)
```

了解网站的提及和引用情况。

### 6. 生成综合报告

基于以上所有数据，参考 `references/report_template.md` 生成完整 Markdown 报告，包含：

1. **总体评分** - 0-100 分
2. **五大维度评估** - 可爬取性、性能、内容优化、内容质量、权威性
3. **搜索引擎表现** - ⭐ 新增部分
   - 收录状态（对应搜索引擎的索引数量）
   - 关键词排名（主要关键词的排名位置）
   - 自然搜索可见度评估
4. **具体问题列表**
5. **改进建议**（按优先级，标注影响维度）
6. **重点提升维度**

报告保存为 `/mnt/user-data/outputs/seo-report-{domain}.md`。

## 评分标准

| 维度 | 权重 | 检测项 |
|------|------|--------|
| 可爬取性 | 20% | robots.txt、meta robots、canonical、sitemap |
| 页面性能 | 20% | HTML大小、资源数量、压缩状态 |
| 内容优化 | 25% | title、description、h1、关键词密度、图片alt |
| 内容质量 | 20% | 字数、结构化程度、可读性 |
| 权威性 | 15% | 外链数量、社交标签、结构化数据 |

## 关键词提取与排名检测

### 提取目标关键词

从页面中提取 3-5 个主要关键词：

1. **优先来源**：
   - Title 标签中的核心词
   - H1 标签内容
   - Meta description 中的重点词

2. **提取原则**：
   - 选择 2-5 个词的短语（非单个词）
   - 排除停用词（的、是、在、for、the 等）
   - 优先业务相关的专业术语

3. **示例**：
   - 页面 title: "Python 数据分析教程 | 从入门到精通"
   - 提取关键词: ["Python 数据分析", "数据分析教程", "Python 教程"]

### 检测排名

对每个关键词分别搜索，检查目标网站的排名：

```
搜索查询："{关键词}"
```

记录：
- 该网站在结果中的位置（第几条）
- 如果前 30 条未出现，标记为"未上榜"
- 如果出现，计算大致排名（第1页=1-10名，第2页=11-20名，第3页=21-30名）

**技巧**：有些关键词可能需要加上品牌词或地域词来找到准确排名：
```
"{关键词}" {域名}
"{关键词}" {品牌名}
```

## 搜索引擎选择规则

脚本会自动识别网站区域（`site_region` 字段），根据结果选择搜索引擎：

### 中国站（site_region: "china"）

**判断依据**：
- 域名以 .cn 结尾
- 包含百度、淘宝、京东等中国网站关键词
- HTML 中 lang="zh-CN" 或 lang="zh-Hans"

**使用搜索引擎**：
1. **百度** - 主要搜索引擎
   - 收录检测：`site:域名`
   - 注意：百度的 site 语法可能不太准确，仅作参考
2. **Bing 国内版** - 辅助搜索引擎
   - 收录检测：`site:域名`

### 国际站（site_region: "international"）

**默认使用**：
- **Google** - 主要搜索引擎
  - 收录检测：`site:域名`
  - 排名检测：准确度高

## 建议与维度关联

每条优化建议都包含 `affects_display` 字段，标明该建议可提升的维度：

| 建议类型 | 影响维度 |
|----------|----------|
| canonical/noindex/lang | 🔍 可爬取性 |
| viewport/HTML压缩/懒加载 | ⚡ 页面性能 |
| title/description/H1/alt | 📝 内容优化 |
| 内容字数/标题层级 | 📊 内容质量 |
| OG标签/Twitter Cards/结构化数据/外链 | 🔗 权威性 |

## 输出格式

报告保存为 `seo-report-<domain>.md`，包含可视化评分和可执行的改进清单，每条建议明确标注可提升的维度。
